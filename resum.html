Oque eu aprendi sobre css

O CSS é uma linguagem usada para deixar os sites mais bonitos e organizados. Com ele, dá para mudar as cores, o tipo e o tamanho das letras, o fundo da página e até a posição de cada coisa na tela. Existem maneiras diferentes de colocar o CSS no HTML, mas o jeito mais organizado é usando um arquivo separado, porque facilita muito na hora de fazer alterações em várias páginas ao mesmo tempo. O CSS funciona através de várias regras e seletores, que servem basicamente para escolher quais partes da página vão receber os estilos (como selecionar elementos pelo nome da tag ou por algum atributo). Com isso, aprendi a mudar títulos, parágrafos, tabelas, listas e formulários para deixar a página bem mais agradável visualmente. Também vi como personalizar os links para que eles mudem de estilo quando o mouse passa por cima deles. Além de mudar o visual parado da página, entendi que o CSS também serve para criar efeitos e animações simples, deixando a navegação mais suave quando mexemos nos botões e menus. Outro ponto importante é a capacidade do site de se adaptar sozinho a qualquer tamanho de ecrã, seja no computador, no tablet ou no telemóvel, usando regras que mudam o estilo dependendo da largura da tela do aparelho. Para definir o tamanho dos textos e dos blocos, vi que não precisamos usar só os pixels (que são fixos), mas podemos usar tamanhos baseados em percentagens, o que ajuda muito na acessibilidade do site. Por fim, o capítulo mostrou que existem estilos prontos na internet criados por outras pessoas. Eles ajudam a adiantar muito o trabalho do dia a dia, porque nos deixam montar páginas bonitas em poucos minutos, sem ter de escrever todo o código do zero. No final, o livro explicou sobre a 'cascata' e a importância de cada regra, que servem para o navegador decidir qual estilo deve aplicar quando o código tem ordens que entram em conflito. Com tudo isso, deu para entender que o CSS ajuda a deixar os sites mais organizados, modernos e fáceis de usar, melhorando muito a experiência de quem acede à página.

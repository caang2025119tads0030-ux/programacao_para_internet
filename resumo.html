Capítulo 2 - Estruturando páginas Web com HTML

A funcão do CSS, da linguagem de programação JavaScript e do HTML é de suma importância para o desenvolvimento de um bom frontend. Cada uma dessas tecnologias possui uma função específica. O HTML é responsável pela estrutura da página, o CSS pela aparência visual e o JavaScript pela interatividade. Quando falamos sobre HTML, estamos tratando da toda a base de toda página web, pois ele organiza o conteúdo para ser interpretado corretamente pelo navegador e da aspecto de uma boa interatividade e organização.

2.1 INTRODUÇÃO

Existiram várias versões do HTML em toda sua jornada até o que utilizamos atualmente. Cada nova versão trouxe melhorias, correções e novas funcionalidades, permitindo a criação de páginas cada vez mais modernas, organizadas e compatíveis com diferentes dispositivos e navegadores. Com o avanço da tecnologia, surgiu também a necessidade de tornar os sites mais acessíveis, rápidos e responsivos, ou seja, adaptáveis a celulares, tablets e computadores. Além disso, as novas versões do HTML passaram a oferecer melhor suporte para multimídia, como áudios e vídeos, sem a necessidade de plugins externos. Essas evoluções também contribuíram para uma melhor organização do código, facilitando a manutenção e o trabalho dos desenvolvedores.

VERSÕES:

HTML< Essa primeira versão foi conhecida em 1992 baseada na proposta de Tim Berners-Lee, considerado o criador da web.

HTML 4.0 Foi lançado em 1997 e trouxe muito mais melhorias e isso e muito importantes, permitindo uma melhor integração com folhas de estilo.

XHTML 1.0 Foi baseado em XML e exigia regras mais rígidas na escrita do código.

HTML5 É a versão atual e foi lançada em 2014, trazendo novos recursos para a construção de páginas modernas. e dando uma melhoria nesse pàginas com um objetivo de alcançar melhorias.

O HTML é mantido pela World Wide Web. Porém, devido à constante evolução da web, existe também o WHATWG, responsável por acompanhar e atualizar os padrões da linguagem. que são fundamentais para melhorar esse padrão de linguagem de uma forma que vai ajudar na melhoria de novas versões mais atualizadas.

2.1.2 USANDO ELEMENTOS E ATRIBUTOS

os elementos e atributos que compõem a estrutura da página web em HTML. Esses elementos são fundamentais para organizar o conteúdo de forma mais clara e lógica, isso facilitando tanto a leitura do usuário quanto a interpretação pelo navegador. Além disso, o uso correto desses elementos contribui para a acessibilidade e para a otimização dos sites.

Cada um desses componentes que faz parte de uma página web é estruturado através de tags. Essas tags definem elementos e suas funções, como títulos, parágrafos e listas. Normalmente, possuem uma tag de abertura e uma de fechamento, envolvendo o conteúdos que será exibido. Elas são escritas utilizando sinais de menor que (<) e maior que (>), formando a base da linguagem HTML. A correta utilização dessas tags é essencial para garantir que o conteúdo seja exibido de forma adequada em diferentes navegadores.

Usa-se o DOCTYPE para especificar a versão correta do HTML utilizada no documento, garantindo que o navegador interprete o código corretamente. Dentro do documento, temos a tag html, que representa o elemento principal da página. Em seguida, temos o head (cabeçalho), que contém informações como título, configurações e metadados, e o body (corpo), onde fica todo o conteúdo visível para o usuário. Essa organização é fundamental para o bom funcionamento da página.

3. ELEMENTOS DE CONTEÚDO E ORGANIZAÇÃO DE DADOS

Após compreendermos a estrutura básica e a história do HTML, é necessário aprofundar nos elementos que permitem organizar a informação de forma hierárquica e legível. O HTML oferece uma vasta gama de tags específicas para diferentes tipos de texto, garantindo que o navegador e os motores de busca entendam exatamente o que é cada parte do conteúdo. Sem o uso de CSS, a organização visual depende inteiramente da escolha correta destas tags.

3.1 TÍTULOS TEXTUAL

A organização de um texto começa pelos seus títulos. O HTML disponibiliza seis níveis de cabeçalhos, do <h1> ao <h6>. O uso correto destes níveis cria um "sumário" invisível para o navegador, ajudando na acessibilidade para leitores de tela. Além dos títulos, utilizamos o elemento <p> para definir parágrafos, que são a unidade básica de qualquer texto informativo na web.

Strong Este elemento é utilizado para indicar que um texto possui forte importância ou urgência. Visualmente, o navegador exibe-o em negrito por padrão.
